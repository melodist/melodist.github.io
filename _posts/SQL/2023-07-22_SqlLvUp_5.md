# 반복문
## 반복문 의존증
- SQL에는 반복문이 없음
- SQL을 적용하기 힘든 작업에 무리하게 SQL을 사용할 필요는 없음
- 미들웨어 또는 O/R 맵퍼 등의 프레임워크가 내부적으로 반복계 코드를 사용하기도 함
- 반복문 의존증 예시
  - 온라인 처리에서 레코드 하나씩 접근하는 SELECT 구문을 반복 사용
  - 배치 처리에서 레코드를 하나씩 호스트 언어에서 처리하고 테이블에 갱신
